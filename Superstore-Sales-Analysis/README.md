# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes Superstore sales data using Power BI.This is project shows the analysis with a simple dashboard. 

## Objectives
- Analyze sales performance
- Identify profitable categories
- Evaluate regional performance
- Track sales trends

## Tools Used
- Power BI
- Excel/CSV Dataset

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Customers

## Dashboard Features
- Sales by Category, Regions
- Monthly Sales Trend
- Profit by Region
- Interactive Filters

## Insights
- Consumer segment drives most sales.
- West region is the best-performing region.
- Phones, Chairs, and Storage are the highest-selling products.
- Fasteners, Labels, and Envelopes contribute the least revenue.
- Sales show a growing trend from 2014-2017.
- Revenue is geographically diversified across the U.S.
- Overall profit margin is around 12-13%.

## Recommandations
- Focus marketing campaigns on Consumer customers.
- Develop loyalty programs for Corporate clients.
- Create targeted promotions to increase Home Office purchases.
